## Selected Projects

### Job Search Workflow Tools

Two connected projects for turning job discovery into a more structured, human-in-the-loop workflow.

- [Search Crawler](https://github.com/VanessaLin9/crawler)  
  `Python` · `Google Sheets` · `GitHub Actions` · `SMTP`  
  Daily crawler for 104, Cake, and Yourator. It collects job listings, normalizes structured fields, dedupes by URL, syncs results to Google Sheets, and sends email summaries.

- [JD Saver for Google Sheets](https://github.com/VanessaLin9/jd-automation)  
  `JavaScript` · `Chrome Extension` · `Google OAuth` · `Google Sheets API`  
  Chrome extension for saving a job description page directly into a user-owned Google Sheet after reviewing it. Available on the [Chrome Web Store](https://chromewebstore.google.com/detail/jd-saver-for-google-sheet/ipcamokmondmagfhndmifhdeldppekni), with a [write-up](https://medium.com/@vanessa7591/%E6%88%91%E6%B2%92%E6%9C%89%E5%81%9A-auto-apply-bot-%E6%88%91%E5%81%9A%E7%9A%84%E6%98%AF%E4%B8%80%E5%A5%97-human-in-the-loop-%E7%9A%84%E6%B1%82%E8%81%B7-workflow-7b5d1f6c50d7).
